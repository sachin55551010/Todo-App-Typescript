# 📝 TypeScript Task Manager App

A simple full-stack task management application built using **TypeScript** on both frontend and backend. This project focuses on learning TypeScript practically by applying it in a real-world application.

---

## 🚀 Features

- Add, update, and delete tasks
- Mark tasks as completed or pending
- Clean and minimal UI
- Full TypeScript support (frontend + backend)
- REST API integration
- Scalable project structure

---

## 🛠️ Tech Stack

### Frontend
- React
- TypeScript
- Vite

### Backend
- Node.js
- Express
- TypeScript

---

## 📚 What I Learned

- Using **TypeScript in real-world projects**
- Creating and managing **interfaces & types**
- Handling **API requests with type safety**
- Structuring a scalable backend with TypeScript
- Difference between **development (ts-node)** and **production (compiled JS)**
- Writing cleaner, predictable, and maintainable code
